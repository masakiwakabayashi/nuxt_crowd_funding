# 権限一覧（機能別 × ロール）

このドキュメントは、アプリ内の主要機能とロールごとの操作可否をまとめたものです。プロダクト全体を俯瞰したいときはセクション単位の表を参照してください。

## Legend
| 記号 | 意味 |
| --- | --- |
| ✅ | 実行可能 |
| 🔼 | 担当分（自分に割り当てられたプロジェクト / 納品）に限定して可能 |
| ❌ | 実行不可 |
| ⭐ | Admin のみ実行可能 |
